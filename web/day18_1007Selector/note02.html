CSS的语法

选择器 声明块
选择器：
    -通过选择器可以选中页面中指定的元素，并且将声明块中的样式应用到选择器对应的元素上

声明块：
    -声明块紧跟在选择器的后边，使用一对{}括起来，声明块中实际上就是一组一组的名值对结构，这一组一组的名值对称之为声明， 多个声明之间使用：隔开，声明的样式名和样式值之间使用：来连接
可以用<style type="text/css"></style>标签来写css样式
也可以用 .css 文件来写样式 （推荐）

块和内联

块元素

<div>就是一个块元素，所谓块元素就是会独占一行的元素，无论他的内容有多少都会独占一行。
div这个标签是没有任何语义的，就是一个纯粹的块元素，并且不会为它里边的元素设置任何的默认样式
div元素主要是用来对页面进行布局的
<p>也是一个块元素
h1 h2 。。。 也是块元素

内联元素

<span>是一个内联元素（行内元素），没有任何语义，专门用来选中文字，然后为文字设置样式
所谓的行内元素，指的是只占自身大小的元素，不会占用一行
常见的内联元素：a、img、iframe、span
块元素主要用来做页面布局，内联元素主要用来选中文本设置样式，
一般情况下只使用块元素去包含内联元素，而不会使用内联元素去包含一个块元素
a标签可以包含任意元素，除了本身；p元素不能包含任何块元素

常用的选择器

1、元素选择器
  -作用： 通过元素选择器可以选择页面中的所有元素
  -语法：标签名 { xxx：xxx }

2、id选择器： 通过元素的id属性值选中唯一的一个元素
  -语法： #id属性值 { xxx：xxx}

3、类选择器
 通过元素的class属性选中一组元素
 语法： .class属性值{ xxx：xxx }
可以为元素设置class属性，拥有相同class属性值的元素，可以称之为一组元素
可以同时为一个元素设置多个class属性值，多个值之间使用空格隔开

4、选择器分组 -- , 分隔
  -通过选择器分组可以同时选中多个选择器对应的元素
  -语法： 选择器1,选择器2,选择器N{ xxx：xxx}

5、通配选择器
  -可以用来选中页面中的所有元素
  -语法：* { xxx：xxx }

6、复合选择器（交集选择器） --不用分隔
  -作用：可以选中同时满足多个选择器的元素
  -语法：选择器1选择器2选择器N { xxx：xxx }
对于id选择器来说，不建议使用复合选择器

7、后代元素选择器 -- 空格 分隔
  -作用：选定元素的指定后代元素
  -语法：祖先元素 后代元素 { xxx：xxx }

8、子元素选择器 > 分隔
  -作用：选中指定父元素的指定子元素
  -语法： 父元素 > 子元素 { xxx：xxx }

9、伪类选择器
伪类专门用来表示元素的一种的特殊的状态，
-比如访问过的超链接，比如普通的超链接，比如获取焦点的文本框
-当我们需要为处在这些特殊状态的元素设置样式时，就可以使用伪类给链接定义样式：
浏览器是通过历史浏览记录来判断一个链接是否访问过，由于涉及到用户的隐私问题，所以使用visited伪类只能设置字体颜色
  - a:link { xxx：xxx } --1.正常链接
  - a:visited { xxx：xxx } --2.访问过的链接
  - a:hover { xxx：xxx } --3.鼠标滑过的链接
  - a:active { xxx：xxx } --4.正在点击的链接
  - 标签：focus { xxx：xxx } --获取焦点
  - 标签：before { xxx：xxx } --指定元素前
  - 标签：after { xxx：xxx } --指定元素后
  - 标签：：selection { xxx：xxx } --选中的元素
标签:hover {} 和 标签:active {} 也可以为其他元素设置

10、伪元素选择器
使用伪元素来表示元素中的一些特殊的位置
  - 标签：first-letter { xxx：xxx } --设置指定标签中的第一个字母的样式
  - 标签：first-line { xxx：xxx } --设置指定标签中的第一行的样式
  - 标签：before { content：xxx } --设置内容紧跟标签头部
  - 标签：after { content：xxx } --设置内容紧跟标签尾部

11、属性选择器
  -作用：可以根据元素中的属性名或属性值来选取指定元素
  -语法1： 标签[属性名] { xxx：xxx } --选定属性名的元素
  -语法2： 标签[属性名="属性值"] { xxx：xxx } --选定"属性值"的元素
  -语法3： 标签^[属性名="值"] { xxx：xxx } --选定以该 值 为开头的属性值元素
  -语法4： 标签$[属性名="值"] { xxx：xxx } --选定以该 值 为结尾的属性值元素
  -语法5： 标签*[属性名="值"] { xxx：xxx } --选定以包含 值 的属性值元素
title属性，这个属性可以给任何标签指定，当鼠标移入到元素上时，元素中的title属性的值将会作为提示文字显示

12、子元素选择器
  :first-child { xxx：xxx } --选择第一个子标签
  :last-child { xxx：xxx } --选择最后一个子标签
  :nth-child(数字) { xxx：xxx } --选择指定位置的子元素，even表示偶数位置的标签，odd表示奇数位置的标签
  :first-of-type、 :last-of-type、 :nth-of-type --选择指定类型的指定位置的元素

13、兄弟元素选择器
后一个兄弟元素选择器 + 分隔
  -作用：可以选中一个元素后紧跟着的指定的兄弟元素
  -语法：前一个 + 后一个 { xxx：xxx }

后边的所有兄弟元素 ~ 分隔
  -作用：可以选中一个元素后紧跟着的指定的所有兄弟元素
  -语法：前一个 ~ 后一个 { xxx：xxx }

14、否定伪类
  -作用：可以从已选中的元素中剔除出某些元素
  -语法： 标签:not(选择器) { xxx：xxx } --剔除含有括号内元素的标签

样式的继承

在CSS中，祖先元素上的样式，会被他的后代元素所继承
利用继承，可以将一些基本的样式设置给祖先元素，这样所有的后代元素也会自动继承这些样式
但并不是所有的样式都会被子元素所继承，比如：背景相关的样式都不会被继承

选择器的优先级

当使用不同的选择器，选中同一个元素时并且设置相同的样式时，这时样式之间就产生了冲突，最终到底采用哪个选择器定义的样式
由选择器的优先级（权重）决定，优先级高的优先显示。
优先级规则：
  -内联样式：优先级 1000
  -id选择器： 优先级 100
  -类和伪类： 优先级 10
  -元素选择器： 优先级 1
  -通配*： 优先级 0
  -继承的样式： 没有优先级
当选择器中包含多种选择器时，需要将多种选择器的优先级相加然后再比较
但是注意：选择器优先级计算不会超过他的最大的数量级，如果选择器的优先级一样，则会使用靠后的样式
并集选择器的优先级是单独计算 ，eg：div , p , #id , .hello { xxx:xxx }
可以在样式的最后，添加一个!important，则此时该样式将会获得一个最高的优先级，将会优先于所有的样式显示，甚至超过内联样式，但在开发中尽量避免使用！important

元素之间的关系

 父元素;直接包含元素的元素
 子元素：直接被父元素包含额元素
 祖先元素：直接或间接包含后代元素的元素，父元素也是祖先元素
 后代元素：直接或间接被祖先元素包含的元素，子元素也是后代元素
 兄弟元素：拥有相同父元素的元素

文本标签

p标签中的元素
1、<em>和<strong> -这两个标签都表示一个强调的内容
  <em>主要表示语气上的强调，em在浏览器中默认使用斜体显示
  <strong>表示强调的内容，比em更强烈，默认使用粗体显示

2、<i>和<b> -这两个标签和em、strong标签样式一样，但h5规范中规定，对于不需要着重的内容而是单纯的加粗或斜体，就可以用i和b标签
  <i>斜体，没语义，在浏览器中默认使用斜体显示
  <b>粗体，没语义，默认使用粗体显示

3、<small> -这个标签的内容会比他的 父元素的文字小一些 ，在h5中使用small标签来表示一些细则一类的内容。比如：合同小字，网站的版权声明都可以放到small中

4、<cite> -网页中所有的加书名号的内容都可以使用cite标签，表示参考的内容，比如 书名、歌名、电影名

5、<q> -表示一个短的引用（行内引用），q标签引用的内容，浏览器会默认加上引号
  <blockquote> -表示一个长引用（块级引用）

6、<sup>和<sub> -表示上标和下标，如 231 和 H2O

7、<del>和<ins> -表示删除和插入，如 del自动添加删除线 、 ins会自动添加下划线

8、<pre>和<code> -表示pre是一个预格式标签，会将代码中的格式保存，不会忽略多个空格；code专门用来表示一段代码；一般结合使用pre和code来表示一段代码

列表

在HTML可以创建列表，在网页中一共有三种列表：1、无序列表；2、有序列表；3、定义列表；
无序列表： <ul>和<li> --块元素；格式 <ul style="list-style:none;"> <li> </li> </ul>
有序列表：<ol>和<li> --块元素；格式 <ol> <li> </li> </ol>
列表之间是可以互相嵌套的 如： 
        
            <ul>
                <li>
                    <ol>
                        <li>
                        </li>
                    </ol>
                </li>
            </ul>

单位

长度单位：
    像素 px
        - 像素是网页中使用得最多的一个单位，显示效果越好，像素就越小，反之像素越大
    百分比 xx%
        - 将单位设置成%的形式，浏览器将会根据父元素的样式来计算该值。在创建一个自适应的页面时常用百分比作为单位
    em
        - em和百分比类似，它是相对于当前元素的字体大小来计算的，1em = 1font-size；当字体发生改变时，em也会随之改变

颜色单位：
  在CSS可以直接使用颜色的单词来表示不同的颜色
    红色：red
    蓝色：blue
    旅社：green
  也可以使用RGB值来表示不同的颜色
    -所谓RGB值指的是通过Red Green Blue三元色的不同浓度，来表示出不同的颜色
    -例子：rgb（红色的浓度，绿色的浓度，蓝色的浓度）
     -颜色的浓度需要一个0-255之间的值，255表示最大，0表示没有
     -浓度也可以采用百分比来设置，需要设置一个0% - 100%来表示0-255之间的数字
